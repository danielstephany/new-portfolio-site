Fitsii

Fitsii

Fitsii is a project that I designed to give myself a bit of practice using angluar.js. Fitsii is a single page web app that uses angulars router to change between different views and set the appropriate controller. I set up a mock database for angular to pull data from so that the information on each view can be dynamically generated using angulars $http service. I also used filtering and ordering in the list views to organize the data on the page. Along with this project I used gulp to handle my tasks and streamline my workflow by having it concat and minimize scripts, compile and minimize Sass, minify html, and place everything in a dist folder for distribution.

Ritualform

Ritualforms

Ritualforms.com is a mobile responsive Portfolio site I designed and built for a local Milwaukee tatoo artist and painter. My client wanted a very simple site that was easy to use, navigate, and post photos of his work on. The photos on this site are pulled from different galleries on flickr that he posts to from his instagram account. To get the images from flickr I used jQuery to perform an ajax call, then I filtered and posted the photos using javascript. I also added a light box effect that I coded with jQuery.
